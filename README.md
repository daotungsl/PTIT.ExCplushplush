- Ôn tập và rèn luyện tư duy thuật toán.
- Thành thạo cú pháp và kỹ thuật lập trình C++ cơ bản đến nâng cao.
- Làm quen với các bài tập phổ biến: cấu trúc lặp, mảng, chuỗi, hàm, đệ quy, struct, v.v.
